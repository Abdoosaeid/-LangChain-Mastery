## 🛠️ Core Concepts Covered
This repository tracks progress through the essential building blocks of the framework:

* **Prompts:** Managing and optimizing LLM inputs.
* **Models:** Interfacing with LLMs (OpenAI, Anthropic, HuggingFace).
* **Memory:** Giving your bots a "brain" to remember past interactions.
* **Chains:** Linking multiple components together for complex workflows.
* **Indexes (RAG):** Connecting LLMs to your private data for grounded answers.
* **Agents:** Enabling the LLM to use external tools like Google Search or Calculators.
